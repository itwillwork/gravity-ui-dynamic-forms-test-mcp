# String TextArea Control

## Description

Multi-line text input control for longer string content. Supports auto-resize, character limits, and provides ample space for descriptions, comments, or other long-form text.

## Configuration

**Spec Type:** `StringSpec`  
**viewSpec.type:** `"textarea"`

## Properties

| Property | Type | Description |
|----------|------|-------------|
| minLength | `integer` | Minimum text length |
| maxLength | `integer` | Maximum text length |
| required | `boolean` | Whether field is required |
| defaultValue | `string` | Initial value |
| inputProps | `object` | Props passed to underlying TextArea component |

## Common inputProps

- **rows**: `number` - Number of visible text rows
- **minRows**: `number` - Minimum rows (for auto-resize)
- **maxRows**: `number` - Maximum rows (for auto-resize)
- **size**: `"s"` \| `"m"` \| `"l"` \| `"xl"` - Control size
- **hasClear**: `boolean` - Show clear button
- **disabled**: `boolean` - Disable the control

## Example - Basic TextArea

```json
{
  "type": "string",
  "required": false,
  "maxLength": 500,
  "viewSpec": {
    "type": "textarea",
    "layoutTitle": "Description",
    "layoutDescription": "Provide a brief description (max 500 characters)",
    "placeholder": "Enter description here...",
    "inputProps": {
      "rows": 5,
      "size": "l",
      "hasClear": true
    }
  }
}
```
